Meu Projeto Docker Desafio DIO

Desafio feito para o curso de Docker Fundamentals que consiste em editar o arquivo yml do docker compose, criando uma página html que subirá em um volume docker e será usada no container da aplicação.

A aplicação é um servidor apache que disponibiliza a aplicação web na porta 80:80 e pode ser acessada através do IP que está rodando o container da aplicação
